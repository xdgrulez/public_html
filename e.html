My AmigaE software

This page contains stuff by me written in AmigaE by Wouter van Oortmerssen. A fantastic language for Amiga system programming, by the way.

EasyPlayer

EasyPlayer is a sophisticated graphical user interface for any (you define it!) shell-based music player running on AmigaOS-compatible systems.

Current version: 1.24 (30.5.2004), EasyPlayer.lha, EasyPlayer.readme

Guideance

Guideance is an AmigaGuide-viewer with support for the XPK compression system, and can also act as a frontend for the alternative AmigaGuide-viewer PowerGuide (as used on the Aminet CDs).

Current version: 1.0 (3.11.1996) Guideance.lha, Guideance.readme
