9.17 Dags2

This section explains the Dags2 output functor.

  • identifier: output.dags2
  • open method: opens a window displaying the dags corresponding to all dimensions using a graph principle (excluding e.g. the lex dimension). The individual dags are drawn as by the Dag2 output functor (Dag12).
  • close method: closes all of these windows and resets the comparisons (i.e. the next output does not compare).
