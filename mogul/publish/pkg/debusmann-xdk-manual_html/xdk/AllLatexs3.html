9.8 AllLatexs3

This section explains the AllLatexs3 output functor.

  • identifier: output.allLatexs3
  • open method: prints the dags corresponding to all dimensions (including e.g. the lex dimension), not only the dimensions using the graph principle (see Latexs3). The individual dags are printed as by the Latex output functor (Latex3).
  • close method: does nothing
