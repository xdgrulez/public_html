9.32 XML2

This section explains the XML2 output functor.

  • identifier: output.xml2
  • open method: prints an XML file containing all known edges (and known edge labels and node labels) of the directed acyclic graph on the output dimension to stdout. The corresponding DTD can be found in Extras/statistics.dtd (starting from the tag graph). (difference to the ordinary XML output functor (XML): also prints the lexical entries corresponding to the nodes.)
  • close method: does nothing
