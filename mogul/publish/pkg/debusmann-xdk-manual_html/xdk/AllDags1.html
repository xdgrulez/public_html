9.2 AllDags1

This section explains the AllDags1 output functor.

  • identifier: output.allDags1
  • open method: opens a window displaying the dags corresponding to all dimensions (including e.g. the lex dimension), not only the dimensions using the graph principle (see Dags1). The individual dags are drawn as by the Dag1 output functor (Dag11)
  • close method: closes all of these windows.
