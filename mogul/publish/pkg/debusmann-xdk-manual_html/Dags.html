9.15 Dags

This section explains the Dags output functor.

  • identifier: output.dags
  • open method: opens a window displaying the dags corresponding to all dimensions using a graph principle (excluding e.g. the lex dimension). The individual dags are drawn as by the Dag output functor (Dag1)
  • close method: closes all of these windows.
