9.4 AllDags3

This section explains the AllDags3 output functor.

  • identifier: output.allDags3
  • open method: opens a window displaying the dags corresponding to all dimensions (including e.g. the lex dimension), not only the dimensions using the graph principle (see Dags3). The individual dags are drawn as by the Dag1 output functor (Dag13)
  • close method: closes all of these windows.
