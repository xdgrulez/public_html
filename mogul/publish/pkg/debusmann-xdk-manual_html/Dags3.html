9.18 Dags3

This section explains the Dags3 output functor.

  • identifier: output.dags3
  • open method: opens a window displaying the dags corresponding to all dimensions using the graph principle (excluding e.g. the lex dimension). The individual dags are drawn as by the Dag3 output functor (Dag13).
  • close method: closes all of these windows and resets the comparisons (i.e. the next output does not compare).
