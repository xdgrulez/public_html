9.3 AllDags2

This section explains the AllDags2 output functor.

  • identifier: output.allDags2
  • open method: opens a window displaying the dags corresponding to all dimensions (including e.g. the lex dimension), not only the dimensions using the graph principle (see Dags2). The individual dags are drawn as by the Dag1 output functor (Dag12)
  • close method: closes all of these windows.
