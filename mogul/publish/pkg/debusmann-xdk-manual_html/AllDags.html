9.1 AllDags

This section explains the AllDags output functor.

  • identifier: output.allDags
  • open method: opens a window displaying the dags corresponding to all dimensions (including e.g. the lex dimension), not only the dimensions using the graph principle (see Dags). The individual dags are drawn as by the Dag output functor (Dag1)
  • close method: closes all of these windows.
